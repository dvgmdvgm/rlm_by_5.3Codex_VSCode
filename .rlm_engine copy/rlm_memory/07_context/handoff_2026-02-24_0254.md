## ⚠️ Критические заметки

- **OLED тема обязательна**: bg `#0D0D0F`, cards `#18181B`, CTA `#2563EB`. Никаких белых фонов
- **`{% trans %}`** — ВСЕ строки в шаблонах. JS: `|escapejs`
- **PowerShell only** — никогда не использовать Unix-команды (du, wc, find, grep, sed)
- **i18n**: компиляция через `compile_mo.py`, НЕ `makemessages`
- **CSS cache busting**: обновлять `?v=X.X.X` при изменении CSS
- Последняя попытка запустить скрипт проверки переводов упала из-за SyntaxError в однострочнике — нужно переписать в отдельный файл
